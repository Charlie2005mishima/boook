### IC (Information Coefficient)
---
【定义】
IC表示预测值（如因子值）与未来收益之间的秩相关系数（Rank Correlation），通常使用 Spearman's Rank 相关系数（Spearman's Rank Correlation）来计算。

【计算公式】
- $$IC = Spearman(Rank(因子值),Rank(未来收益))$$
- $$\rho=\frac{\sum_i(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_i(x_i-\bar{x})^2\sum_i(y_i-\bar{y})^2}}$$

【拓展】
- 平均IC（Mean IC）
- $$ICIR=\frac{Mean(IC)}{Std(IC)}$$

【代码】
```python
from scipy.stats import spearmanr
def calculate_ic(factor,return_value):
	ic,_=spearmanr(factor,return_value)
	return ic
```

### IR (information Ratio)
---
【定义】
$$IR=\text{超额收益的年化均值} / \text{跟踪误差}$$
> **超额收益**：策略收益与基准收益之差
> **跟踪误差**：超额收益的标准差，衡量策略收益相对于基准收益的波动率

【计算公式】
$$IR=[\sum_1^n 超额收益_i/n]/\sqrt{\sum_i^n(超额收益_i-超额收益均值)^2/(n-1)}$$

【代码】
```python
import numpy as np
mean_excess_return = np.mean(excess_returns)
mean_excess_return = np.std(excess_returns, ddof=1) # 使用样本标准差
IR = mean_excess_return / mean_excess_return
```

>入库标准一般：平均ic高于0.03，icir大于0.5，自相关性低于0.7

### 中心化
---
以行业和市值双中性为例：
- **第1步：初级排序：首先**按行业**对全市场股票进行划分。这是第一层。
- **第2步：次级排序与分组**：在**每一个行业内部**，将股票按照**市值**从小到大排序，并分为**N个组**（例如，5个市值组）。
- **第3步：应用目标因子进行最终分组**：在**每一个“篮子”内部**（例如，“银行行业-最小市值组”这个篮子里的所有股票），再按照我们的**目标因子（隔夜涨跌幅因子）** 从小到大进行排序。
- **第4步：合并构建组合**

### 正交化处理
---
$$因子_1 = β × 因子_2 + 因子_3$$
- `β × 因子2`：代表了因子1中可以被因子2解释或"预测"的部分。
- `因子3`（即残差）：代表了因子1中**独立于因子2**的、独特的、无法被预测的部分。
