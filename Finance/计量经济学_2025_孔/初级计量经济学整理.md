## 二、一元线性回归模型的参数估计
- 经典假设【背】
	1. **模型**：回归模型是正确设定的
	2. **样本变异、方差有限**：解释变量 $X$在所抽取的样本中具有变异性，而且随着样本容量的无限增加，解释变量$X$的样本方差趋于一个非零的有限常数
	3. **零均值且无关**：随机误差项条件均值零且与自变量无相关性
	4. **同方差、序列不相关**：随机误差项 $\mu_i$具有同方差性与不序列相关性
	5. **正态分布**：随机误差项 $\mu_i$服从正态分布
- 普通最小二乘法
	-  $\begin{cases}\sum e_i=0\\\sum e_iX_i=0\end{cases}$
	- $\begin{cases}\sum (Y_i-\hat{\beta_0}-\hat{\beta}_1X_i)=0\\\sum (Y_i-\hat{\beta_0}-\hat{\beta}_1X_i)X_i=0\end{cases}$
	- $\begin{cases}\hat{\beta}_1 = \frac{n\sum X_iY_i-\sum X_i \sum Y_i}{n\sum X_i^2-(\sum X_i)^2}=\frac{cov(X,Y)}{var(X)}=\rho_{XY}\frac{\sigma_Y}{\sigma_X}\\\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}\end{cases}$
	- $\hat{\beta}_1 = \frac{\sum x_i y_i}{\sum x_i^2}, \quad \hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X},\quad \hat{y}_i=\hat{\beta}_1x_i$ 
	- $\begin{cases}\hat{\beta}_0=\bar{Y}-\hat{\beta}_1\bar{X}=\beta_0+\bar{\epsilon}-A\bar{X}\\\hat{\beta}_1=\frac{\sum x_iy_i}{\sum x_i^2}=\beta_{1}+ \frac{\sum_{i=1}^nx_{i}\varepsilon_{i}}{S_{xx}}\end{cases}$ 
- 最大似然估计
	- $\begin{cases}\hat{\beta}_0=\bar{Y}-\hat{\beta}_1\bar{X}=\beta_0+\frac{\sum\epsilon_{i}}{n}- \frac{\sum x_{i}\varepsilon _{i}}{\sum x_{i}^2}\bar{X}\\\hat{\beta}_1=\frac{\sum x_iy_i}{\sum x_i^2}=\beta_{1}+ \frac{\sum x_{i}\varepsilon_{i}}{\sum x_{i}^2}\end{cases}$
	- $\sigma_{ML}^2=\frac{\sum e_i^2}{n}$ 
## 三、最小二乘估计量的统计性质
- Guass-Markow定理【证明】
	- **线性性**：即估计量$\hat{\beta}_0,\hat{\beta}_1$是$Y_i$线性组合
	- **无偏性**：即估计量的均值$\hat{\beta}_0,\hat{\beta}_1$等于总体回归参数真值$\beta_0,\beta_1$ 
	- **有效性（最小方差）**：即在所有线性无偏估计量中，最小二乘法得到的估计量具有最小方差
	- notes. 只要严格参照变量的数学特征进行计算即可
- 估计量的分布
	- $\hat{\beta}_1 \sim N\left(\beta_1, \frac{\sigma^2}{\sum x_i^2}\right), \quad \hat{\beta}_0 \sim N\left(\beta_0, \frac{\sigma^2 \sum X_i^2}{n \sum x_i^2}\right)$
- 随机误差项方差的估计
	- $\hat{\sigma}^2 = \frac{\sum e_i^2}{n-2}$ 
## 四、一元线性回归模型的基本统计检验
- 拟合优度检验
	- 拟合优度 $R^2= \frac{ESS}{TSS}$
- 变量显著性检验（ t 检验）
	- $t_{\hat{\beta}_1} = \frac{\hat{\beta}_1}{S_{\hat{\beta}_1}} \sim t(n-2)$ 
- 参数的置信区间
	- 在置信水平 $1-\alpha$下：$\hat{\beta}_1 \pm t_{\alpha/2} \cdot S_{\hat{\beta}_1}$ 
## 五、预测问题
- 均值预测 ：$\hat{Y}_0\sim N(Y_0,\sigma^2(\frac{1}{n} + \frac{(X_0 - \bar{X})^2}{\sum x_i^2})$ 
- 区间预测
	- **条件均值** $E(Y|X_0)$ 的置信区间
	- **个别值** $Y_0$ 的预测区间：$\hat{Y}_0-Y_0\sim N(0,\sigma^2(1 + \frac{1}{n} + \frac{(X_0 - \bar{X})^2}{\sum x_i^2}))$
# chap3 多元线性回归模型
## 3.1.多元线性回归模型
- 模型设定：$Y_i = \beta_0 + \beta_1 X_{i1} + \beta_2 X_{i2} + \cdots + \beta_k X_{ik} + \mu_i$ 
- 经典假设
	1. 回归模型是正确设定的
	2. 解释变量在所抽取的样本中具有==变异性==，且变量之间不存在==严格线性相关==性，随着样本量的无限增加，$X'X/n$依概率收敛于一可逆有限常矩阵$Q$
	3. **条件零均值**：给定任意解释变量，随机==误差项均值为零==
	4. **同方差性**：随机扰动项具有==条件同方差==与==序列不相关性==
	5. **正态性**：随机扰动项服从零均值同方差的正态分布
## 3.2. 参数估计
- 最小二乘法
	- OLS估计量：$\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Y}$ 
	- **残差性质**：$\mathbf{X}'\mathbf{e} = 0$或$\begin{cases}\sum e_i=0\\\sum_i X_{ij}e_i=0\end{cases}$ 
- 最大似然估计
	- $\hat{\boldsymbol{\beta}}_{ML} = \hat{\boldsymbol{\beta}}_{OLS}$
	- $\hat{\sigma}^2_{ML} = \frac{\mathbf{e}'\mathbf{e}}{n}$ 
- 估计量性质
	1. **线性性**：$\hat{\boldsymbol{\beta}} = \mathbf{C}\mathbf{Y}$
	2. **无偏性**：$E(\hat{\boldsymbol{\beta}}) = \boldsymbol{\beta}$
	3. **有效性**（Gauss-Markov定理）：$Cov(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}'\mathbf{X})^{-1}$
	4. **一致性**：$plim\ \hat{\boldsymbol{\beta}} = \boldsymbol{\beta}$
- 方差估计
	- **误差方差的无偏估计**：$\hat{\sigma}^2 = \frac{\mathbf{e}'\mathbf{e}}{n-k-1}$ 
	- **系数估计量的方差**：$Cov(\hat{\beta})=\sigma^2(\mathbf{X'X})^{-1}$ 
## 3.3.统计检验
- 拟合优度检验
	- 可决系数：$R^{2}= \frac{ESS}{TSS}$
	- 调整的可决系数：$\bar{R}^{2}=1- \frac{RSS/(n-k-1)}{TSS/(n-1)}$
	- 信息准则
		- 赤池信息准则AIC：$\ln\left(\frac{\mathbf{e}'\mathbf{e}}{n}\right) + \frac{2(k+1)}{n}$ 
		- SC: $\ln\left(\frac{\mathbf{e}'\mathbf{e}}{n}\right) + \frac{k}{n}\ln n$ 

|     | 平方和                             | 自由度   | 约束                      |
| --- | ------------------------------- | ----- | ----------------------- |
| 回归  | $ESS=\sum(\hat{Y_i}-\bar{Y})^2$ | k     | k+1个参数，加上$\beta_0$被均值捕获 |
| 残差  | $RSS=\sum(Y_i-\hat{Y_i})^2$     | n-k-1 | k+1个离差方程                |
| 总变差 | $TSS=\sum(Y_i-\bar{Y})^2$       | n-1   | 均值一个限制                  |

- 方程显著性检验
	- F 统计量：$F = \frac{ESS/k}{RSS/(n-k-1)} \sim F(k, n-k-1)$ 
	- F 统计量与 $R^2$ 的关系：$F = \frac{R^2/k}{(1-R^2)/(n-k-1)}$ 
- 变量显著性检验
	- t 统计量：$t_j = \frac{\hat{\beta}_j}{se(\hat{\beta}_j)} \sim t(n-k-1)$ 
- 系数置信区间
	- $\hat{\beta}_j \pm t_{\alpha/2}(n-k-1) \cdot se(\hat{\beta}_j)$
## 3.4.预测
- 均值预测
	- 点预测：$\hat{Y}_0 = \mathbf{X}_0\hat{\boldsymbol{\beta}}$
	- 区间预测：$\hat{Y}_0 \pm t_{\alpha/2} \cdot \hat{\sigma} \sqrt{\mathbf{X}_0(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}_0'}$
- 个值预测
	- 预测误差方差：$Var(e_0) = \sigma^2[1 + \mathbf{X}_0(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}_0']$ 
	- 预测区间：$\hat{Y}_0 \pm t_{\alpha/2} \cdot \hat{\sigma} \sqrt{1 + \mathbf{X}_0(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}_0'}$

经济原因复习

- 序列相关性
	- 定义：$Cov(\mu_{i},\mu_{j})=E(\mu_{i}\mu_{j})\ne0$
	- 检验：
		- 图示法
		- 回归检验法：$e_{t}=\beta_{1}e_{t-1}+\beta_{2}e_{t-2}+\dots\beta_{j}e_{t-j}$ 
		- DW 检验法：$D.W.=\frac{\sum_{t=2}^{T}(\tilde{e}_{t}-\tilde{e}_{t-1})^{2}}{\sum_{t=1}^{T}\tilde{e}_{t}^{2}}\approx 2(1-\rho)$ 
		- BGLM 检验法：$\tilde{e}_{t}=\beta_{0}+\beta_{1}X_{t1}+\dots+\beta_{k}X_{tk}+\rho_{1}\tilde{e}_{t-1}+\dots+\rho_{p}\tilde{e}_{t-p}+\varepsilon_t$$LM \approx n \cdot R^{2} \sim \chi^{2}(p)$ 
	- 改进：
		- 广义最小二乘：$\hat{\beta}_{*}=(\mathbf{X}'\Omega^{-1} \mathbf{X})^{-1}\mathbf{X}'\Omega^{-1}\mathbf{Y}$
		- 广义差分：$\Delta Y_{t} = \beta_{0}' + \beta_{1}' \Delta X_{t} + \epsilon_{t}$
			- 普萊斯-溫斯特變換：$Y_{1}^*=\sqrt{ 1-\rho^2 }Y_{1},X_{1j}^*=\sqrt{ 1-\rho^{2} }X_{ij}$ 
		- 对$\rho$进行估计：科克倫-奧科特 (Cochrane-Orcutt) 迭代法
		- 序列相關穩健標準誤法：对方差进行调整
		- 虛假序列相關問題
- 时间序列平稳性
	- 定义：$E(X_t)=\mu,Var(X_t)=\sigma^2,Cov(X_t,X_{t+k})=\gamma_k$ 
	- 检验
		- 平稳性检验
			- Bartlett 定理
			- Q统计量 $Q_{LB}=n(n+2)\sum_{k=1}^m \frac{r_{k}^{2}}{n-k}\sim \chi(m)$
		- 平稳性的单位根检验
			- DF检验：$\Delta X_{t}=\delta X_{t-1}+\mu_{t}$
			- ADF检验：$\begin{cases}\Delta X_{t}&=\delta X_{t-1}+\sum_{i=1}^m\beta _i\Delta X_{t-i}+\mu_{t}\\\Delta X_{t}&=\alpha+\delta X_{t-1}+\sum_{i=1}^m\beta _i\Delta X_{t-i}+\mu_{t}\\\Delta X_{t}&=\alpha+\beta t+\delta X_{t-1}+\sum_{i=1}^m\beta _i\Delta X_{t-i}+\mu_{t}\end{cases}$ 
- 协整
	- 定义：$Z_t = \alpha X^T \sim I(d-b),\quad b>0$
	- 检验
		- 两变量的 Engle–Granger 检验
			- $Y_t \sim I(1), X_t \sim I(1)$：$\Delta e_t = \delta e_{t-1} + \sum_{i=1}^p \theta_i \Delta e_{t-i} + \varepsilon_t$ 
		- 多变量的 Engle - Granger 检验
			- $Z_{t}=\alpha_{0}+\alpha_{1}W_{t}+\alpha_{2}X_{t}+\alpha_{3}Y_{t}+\mu_{t}$
			- $\Delta \mu_t = \delta \mu_{t-1} + \sum_{i=1}^p \theta_i \Delta \mu_{t-i} + \varepsilon_t$
