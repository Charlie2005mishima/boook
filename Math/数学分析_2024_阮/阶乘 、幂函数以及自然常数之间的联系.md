
*   **增长等级：**
    对于任意 $x > 0$，有：$\ln n \ll x^n \ll n! \ll n^n$
    **命题**：$\lim_{n \to \infty} \frac{x^n}{n!} = 0$。这个简单的极限是 $e^x$ 级数收敛性的基础。

*   **斯特林公式**：
    - $n! \sim \sqrt{2\pi n} \left( \frac{n}{e} \right)^n$
    *   **深度理解**：它告诉我们阶乘的本质是一个“带修正项的指数函数”。
    *   **对数形式**：$\ln n! = n \ln n - n + O(\ln n)$。这在统计力学（玻尔兹曼熵）中至关重要。
*   **Gamma定义**：
    - $\Gamma(z) = \int_0^\infty t^{z-1} e^{-t} \, dt,\,\,$对于正整数 $n$，$\Gamma(n+1) = n!,\,\Gamma\left( \frac{1}{2} \right)=\sqrt{ \pi }$
    *   这里直接把 $n!$、指数函数 $e^{-t}$ 和幂函数 $t^{z-1}$ 放在同一个积分方程里。
*   **魏尔斯特拉斯乘积**：
    - $\frac{1}{\Gamma(z)} = ze^{\gamma z} \prod_{n=1}^\infty \left( 1 + \frac{z}{n} \right) e^{-z/n}$ 这里展示了阶乘倒数与 $e$、欧拉常数 $\gamma$ 之间的乘积联系。
*   **指数函数的泰勒展开**：
    - $e^x = \sum_{n=0}^\infty \frac{x^n}{n!}$
*   **比值审敛法**：
    - 在判断级数 $\sum a_n$ 收敛性时，$\frac{a_{n+1}}{a_n}$ 的极限常涉及 $e$。例如：$\lim_{n \to \infty} \frac{\sqrt[n]{n!}}{n} = \frac{1}{e}$。
*   **泊松分布（Poisson Distribution）**：
    - $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
*   **错排问题（Derangements）**：
    在组合数学中，将 $n$ 个元素全错位的排列数 $D_n$ 与 $n!$ 的关系为：$\lim_{n \to \infty} \frac{D_n}{n!} = \frac{1}{e}$
*   **$e$ 的定义极限**：
    $\lim_{n \to \infty} \left( 1 + \frac{x}{n} \right)^n = e^x$
*   **常用的阶乘不等式**：
    $e \left( \frac{n}{e} \right)^n < n! < e \cdot n \left( \frac{n}{e} \right)^n$
    这是通过对 $\ln n! = \sum \ln k$ 进行积分近似（Euler-Maclaurin 求和公式）得到的常用放缩。
*   **沃利斯公式**：
    $\frac{\pi}{2} = \prod_{n=1}^\infty \frac{4n^2}{4n^2-1}$ 它通常是证明斯特林公式中系数是 $\sqrt{2\pi}$ 的关键一步。
*   **高斯积分**：
    $\int_{-\infty}^\infty e^{-x^2} dx = \sqrt{\pi}$


**1. 指数函数和对数函数**
- $e^x = \sum_{k=0}^{n} \frac{x^k}{k!} + o(x^n) = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \dots, \quad x\in\mathbb{R}$
- $\ln(1+x) = \sum_{k=1}^{n} (-1)^{k-1} \frac{x^k}{k} + o(x^n) = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots, \quad |x|<1$
- $a^x = e^{x\ln a} = \sum_{k=0}^{n} \frac{(\ln a)^k}{k!} x^k + o(x^n), \quad a>0, \, x\in\mathbb{R}$
**2. 三角函数**
- $\sin x = \sum_{k=0}^{n} (-1)^k \frac{x^{2k+1}}{(2k+1)!} + o(x^{2n+2}) = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots, \quad x\in\mathbb{R}$
- $\cos x = \sum_{k=0}^{n} (-1)^k \frac{x^{2k}}{(2k)!} + o(x^{2n+1}) = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots, \quad x\in\mathbb{R}$
- $\tan x = x + \frac{x^3}{3} + \frac{2x^5}{15} + o(x^6), \quad |x|<\frac{\pi}{2}$
- $\arctan x = \sum_{k=0}^{n} (-1)^k \frac{x^{2k+1}}{2k+1} + o(x^{2n+2}) = x - \frac{x^3}{3} + \frac{x^5}{5} - \dots, \quad |x|\le 1$
- $\arcsin x = x + \frac{x^3}{6} + \frac{3x^5}{40} + o(x^6), \quad |x|\le 1$
**3. 幂函数（二项式展开）**
$(1+x)^{\alpha} = \sum_{k=0}^{n} \binom{\alpha}{k} x^k + o(x^n), \quad |x|<1$
$\frac{1}{1-x} = \sum_{k=0}^{n} x^k + o(x^n) = 1 + x + x^2 + \dots, \quad |x|<1$
$\frac{1}{1+x} = \sum_{k=0}^{n} (-1)^k x^k + o(x^n), \quad |x|<1$
$\sqrt{1+x} = 1 + \frac{x}{2} - \frac{x^2}{8} + \frac{x^3}{16} - \dots + o(x^n)$
$\frac{1}{\sqrt{1+x}} = 1 - \frac{x}{2} + \frac{3x^2}{8} - \frac{5x^3}{16} + \dots + o(x^n)$
**4. 双曲函数**
$\sinh x = \sum_{k=0}^{n} \frac{x^{2k+1}}{(2k+1)!} + o(x^{2n+2}) = x + \frac{x^3}{3!} + \frac{x^5}{5!} + \dots$
$\cosh x = \sum_{k=0}^{n} \frac{x^{2k}}{(2k)!} + o(x^{2n+1}) = 1 + \frac{x^2}{2!} + \frac{x^4}{4!} + \dots$
**5. 余项形式**
- **佩亚诺（Peano）余项**：$f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(0)}{k!} x^k + o(x^n)$
- **拉格朗日（Lagrange）余项**：$f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(0)}{k!} x^k + \frac{f^{(n+1)}(\theta x)}{(n+1)!} x^{n+1}, \quad 0<\theta<1$
