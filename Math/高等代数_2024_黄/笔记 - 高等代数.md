### 特征值与特征向量的求解
特征值 $\lambda$ 满足特征方程：$\det(A - \lambda I) = 0$, 其中 $I$ 是单位矩阵。这是一个关于 $\lambda$ 的多项式方程（特征多项式）。
对于每个特征值 $\lambda_i$，解齐次线性方程组：$(A - \lambda_i I) \mathbf{v} = \mathbf{0}$, 得到的非零解向量就是对应于 $\lambda_i$ 的特征向量。
设 $\lambda = \alpha + i \beta$（其中 $\alpha, \beta \in \mathbb{R}$，且 $\beta \neq 0$）是一个复数特征值，其对应的特征向量 $\mathbf{v}$ 是一个复向量。通常我们将 $\mathbf{v}$ 写成实部和虚部之和：$\mathbf{v} = \mathbf{u} + i \mathbf{w},$ 其中 $\mathbf{u}$ 和 $\mathbf{w}$ 都是实向量。

## 向量求导
#### 1 标量函数对向量求导
设 $x = (x_1,\dots,x_n)^T$ 为列向量，$a$ 为与 $x$ 同维的列向量，$A$ 为 $n\times n$ 矩阵。

| 函数                        | 导数 $\frac{\partial f}{\partial x}$ | 说明                        |
| ------------------------- | ---------------------------------- | ------------------------- |
| $f = a^T x$               | $a$                                | 线性形式                      |
| $f = x^T a$               | $a$                                | 同上                        |
| $f = x^T x$               | $2x$                               | 向量内积                      |
| $f = x^T A x$             | $(A + A^T)x$                       | 若 $A$ 对称，则为 $2Ax$         |
| $f = a^T x x^T b$         | $(ab^T + ba^T)x$                   | 或写作 $(a^T x)b + (b^T x)a$ |
| $f = (Ax - b)^T (Ax - b)$ | $2A^T (Ax - b)$                    | 最小二乘残差平方和                 |

#### 2 向量函数对向量求导
设 $f: \mathbb{R}^n \to \mathbb{R}^m$，即 $f(x) = (f_1(x),\dots,f_m(x))^T$。其导数是一个 $m\times n$ 矩阵（雅可比矩阵，分母布局）。

| 函数 | 导数 $\frac{\partial f}{\partial x}$ | 说明 |
|------|------|------|
| $f(x) = Ax$ | $A$ | 线性变换 |
| $f(x) = a$（常向量）| $0$ | 零矩阵 |
| $f(x) = x$ | $I$ | 单位阵 |
| $f(x) = x^T A$ | $A^T$ | 注意：$x^T A$ 是行向量 |

#### 3 标量函数对矩阵求导
设 $X$ 为 $m\times n$ 矩阵，$a,b$ 为适当维数的列向量。

| 函数 | 导数 $\frac{\partial f}{\partial X}$ | 说明 |
|------|------|------|
| $f = a^T X b$ | $a b^T$ | 双线性形式 |
| $f = \operatorname{tr}(AX)$ | $A^T$ | 迹运算 |
| $f = \operatorname{tr}(X^T A X)$ | $(A + A^T)X$ | 若 $A$ 对称，则为 $2AX$ |
| $f = \det(X)$ | $\det(X) (X^{-1})^T$ | 行列式（$X$ 可逆） |
| $f = \log\det(X)$ | $(X^{-1})^T$ | 对数行列式 |

#### 4 链式法则
若 $z = g(y)$，$y = f(x)$，则$\frac{\partial z}{\partial x} = \frac{\partial g}{\partial y} \cdot \frac{\partial f}{\partial x}$
注意维度匹配：$\frac{\partial z}{\partial x}$ 为 $1\times n$（若 $z$ 为标量），$\frac{\partial g}{\partial y}$ 为 $1\times m$，$\frac{\partial f}{\partial x}$ 为 $m\times n$。
若涉及向量对向量，则 $\frac{\partial g(y)}{\partial x} = \frac{\partial g}{\partial y} \cdot \frac{\partial y}{\partial x}$（矩阵乘法）。
