15. $\mathbf{X}=(X_{1},\dots,X_{n})\sim U(0,\theta)$，用零无偏估计法求$\theta$的UMVUE
解：
设 $U(\mathbf{X})$ 是任意满足 $E_\theta[U(\mathbf{X})] = 0$ 对所有 $\theta > 0$ 的可测函数。$E_\theta[U(\mathbf{X})] = \int_{0 < x_{(n)} < \theta} U(\mathbf{x}) \cdot \frac{1}{\theta^n} d\mathbf{x} = 0,\quad \forall \theta > 0$
令 $g(t) = E[U(\mathbf{X}) \mid X_{(n)} = t]$，即条件期望（这是 $t$ 的函数）。由 Rao-Blackwell，任何估计量都可以投影到 $T$ 上而不增加方差。那么：$E_\theta[U(\mathbf{X})] = E_\theta[g(T)] = \int_0^\theta g(t) \cdot \frac{n t^{n-1}}{\theta^n} dt = 0,\quad \forall \theta > 0$
两边乘以 $\theta^n$：$\int_0^\theta g(t) n t^{n-1} dt = 0,\quad \forall \theta > 0$
对 $\theta$ 求导（Leibniz法则）：$g(\theta) n \theta^{n-1} = 0 \quad \Rightarrow \quad g(\theta) = 0 \quad \text{a.e.}$
因此，$g(t) = 0$ 几乎处处成立，即：$E[U(\mathbf{X}) \mid T] = 0 \quad \text{a.s.}$
现在计算协方差：$E[\hat{\theta}_0 U(\mathbf{X})] = E\left[ \frac{n+1}{n} T \cdot U(\mathbf{X}) \right] = \frac{n+1}{n} E\left[ T \cdot E[U(\mathbf{X}) \mid T] \right] = \frac{n+1}{n} E[T \cdot 0] = 0$

19. 我来证明在正态分布 $X_1,\dots,X_n \sim N(a,\sigma^2)$ 下，样本均值 $\bar{X}$ 和极差 $R = X_{(n)} - X_{(1)}$ 是独立的。
**步骤1：正交变换**
构造正交矩阵 $A = (a_{ij})_{n \times n}$，其中第一行：$a_{1j} = \frac{1}{\sqrt{n}}, \quad j=1,\dots,n$
令 $\mathbf{Y} = A\mathbf{Z}$，其中 $\mathbf{Z} = (Z_1,\dots,Z_n)^T$，则：
- $Y_1 = \frac{1}{\sqrt{n}}\sum_{i=1}^n Z_i = \sqrt{n}\bar{Z}$
- $Y_2,\dots,Y_n$ 是与 $Y_1$ 独立的标准正态变量（因为正交变换保持独立性）
**步骤2：表达极差**
注意到：$Z_i = \sum_{j=1}^n a_{ji} Y_j = \frac{Y_1}{\sqrt{n}} + \sum_{j=2}^n a_{ji} Y_j$
因此：$Z_{(n)} - Z_{(1)} = \max_i Z_i - \min_i Z_i = \max_i \left(\frac{Y_1}{\sqrt{n}} + W_i\right) - \min_i \left(\frac{Y_1}{\sqrt{n}} + W_i\right)$
其中 $W_i = \sum_{j=2}^n a_{ji} Y_j$ 仅依赖于 $Y_2,\dots,Y_n$。
**步骤3：关键观察**
$Z_{(n)} - Z_{(1)} = \left(\frac{Y_1}{\sqrt{n}} + \max_i W_i\right) - \left(\frac{Y_1}{\sqrt{n}} + \min_i W_i\right) = \max_i W_i - \min_i W_i$ **极差 $R_Z$ 与 $Y_1$ 无关！**
由于 $Y_1$ 与 $(Y_2,\dots,Y_n)$ 独立，而 $R_Z$ 仅依赖于 $(Y_2,\dots,Y_n)$，因此 $Y_1$ 与 $R_Z$ 独立。
又因为 $\bar{Z} = Y_1/\sqrt{n}$，所以 $\bar{Z}$ 与 $R_Z$ 独立。
