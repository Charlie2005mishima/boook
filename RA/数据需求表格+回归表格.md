这是一个简单的先行试验，我们主要根据年评来做文本因子的有效性检验，下面是我们三个模型的设定，请你根据下述内容写State代码。
### 表1：总体回归模型设定 (Regression Specifications)

我们需要构建三个主要的计量方程，分别对应你的三个文本维度。

**通用模型形式**：
$$ Y_{i, t+1} = \alpha + \beta \cdot \text{TextFactor}_{i,t} + \gamma \cdot \text{Controls}_{i,t} + \text{FixedEffects} + \varepsilon_{i,t} $$

#### 模型 I：信息含量与技能 (Skill) -> 预测未来业绩
*此模型用于检验“逻辑密度”和“独特性”是否代表真实能力。*

| 设定项                         | 变量/描述                                                                                             |
| :-------------------------- | :------------------------------------------------------------------------------------------------ |
| **被解释变量 ($Y_{i, t+1}$)**    | **$Alpha_{i, t+1}$** (未来一年的 Fama-French 3/5因子 Alpha) <br> *或者* **$DGTW\_Ret_{i, t+1}$** (特征调整后收益) |
| **核心解释变量 ($X_{i,t}$)**      | **$\text{Logical\_Density}_{i,t}$** (逻辑密度) <br>信息密度<br> **$\text{Uniqueness}_{i,t}$** (观点独特性)     |
| **控制变量 ($Controls_{i,t}$)** | $\dots$                                                                                           |
| **固定效应 (FE)**               | $\mu_t$ (Year FE) + $\theta_s$ (基金固定效应)                                                           |
| **标准误 (SE)**                | Cluster at Fund Level (聚类到基金代码)                                                                   |
| 预期结果 ($\beta$)              | **显著为正 (+)**。逻辑越强、越独特的文本，未来超额收益越高。                                                                |

#### 模型 II：行为偏差 (Bias) -> 预测未来风险/反转
*此模型用于检验“过度自信”和“自利归因”是否导致未来的盲目冒险或业绩崩塌。*

| 设定项                         | 变量/描述                                                                                                                     |
| :-------------------------- | :------------------------------------------------------------------------------------------------------------------------ |
| **被解释变量 ($Y_{i, t+1}$)**    | **$Volatility_{i, t+1}$** (未来波动率) <br> *或者* **$Max\_Drawdown_{i, t+1}$** (最大回撤) <br> *或者* **$Alpha_{i, t+1}$** (用于检验业绩反转) |
| **核心解释变量 ($X_{i,t}$)**      | **$\text{Overconfidence}_{i,t}$** (过度自信) <br> **$\text{Self\_Serving\_Bias}_{i,t}$** (自利归因)<br>短视主义                       |
| **控制变量 ($Controls_{i,t}$)** | $Log\_TNA$, $Expense$, $Turnover$ (很重要), $Past\_Volatility$ (自回归项), $Past\_Ret,\,\dots$                                   |
| **固定效应 (FE)**               | $\mu_t$ (Year FE) + $\theta_s$ (基金固定效应)                                                                                   |
| **标准误 (SE)**                | Cluster at Fund Level                                                                                                     |
| **预期结果 ($\beta$)**          | 对波动率/回撤：**显著为正 (+)** (越自信越浪)。<br> 对 Alpha：**显著为负 (-)** (过度自信导致亏损)。                                                        |

#### 模型 III：代理问题 (Agency) -> 预测资金流 (Flow)
*此模型用于检验“策略性模糊”和“推卸责任”是否会被投资者惩罚（用脚投票）。*

| 设定项                         | 变量/描述                                                                                                      |
| :-------------------------- | :--------------------------------------------------------------------------------------------------------- |
| **被解释变量 ($Y_{i, t+1}$)**    | **$Flow_{i, t+1}$** (未来一年资金净流入率)                                                                           |
| **核心解释变量 ($X_{i,t}$)**      | **$\text{Obfuscation}_{i,t}$** (迷雾/模糊指数) <br> **$\text{Accountability}_{i,t}$** (责任承担，预期符号相反)<br> 样本文      |
| **控制变量 ($Controls_{i,t}$)** | **$Past\_Perf_{i,t}$** (最核心的控制变量), $Log\_TNA$ (规模效应), $Log\_Age$, $Flow_{i,t}$ (流量自相关), $Expense,\, \dots$ |
| **固定效应 (FE)**               | $\mu_t$ (Year FE) + $\theta_s$ (Fund FE)                                                                   |
| **标准误 (SE)**                | Cluster at Fund Level                                                                                      |
| **预期结果 ($\beta$)**          | 对 Obfuscation: **显著为负 (-)** (看不懂就不买)。<br> 对 Accountability: **显著为正 (+)** (担责能挽回信任)。                        |
