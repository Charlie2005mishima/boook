
| 维度             | 变量名称                          | 计算公式                                                                                          | 符号解释                                                | 文献来源                         |
| :------------- | :---------------------------- | :-------------------------------------------------------------------------------------------- | :-------------------------------------------------- | :--------------------------- |
| **I. 信息含量与技能** | **1. 逻辑密度** (Logical Density) | $LD_i = \sum_{w \in D_{caus}} TF\text{-}IDF_{w,i}$                                            | $D_{caus}$：因果连词词典；$TF\text{-}IDF$：词语在报告 $i$ 中的权重。   | Li (2008)                    |
|                | **2. 信息颗粒度** (Granularity)    | $Gran_i = \frac{N_{NUM,i}}{N_{total,i}}$                                                      | $N_{NUM}$：词性标注为数字的词数；$N_{total}$：报告总词数。             | L&M (2016)                   |
|                | **3. 观点独特性** (Uniqueness)     | $Uniq_i = 1 - \frac{\vec{V}_i \cdot \bar{\vec{V}}_M}{\|\vec{V}_i\| \|\bar{\vec{V}}_M\|}$      | $\vec{V}_i$：基金报告向量；$\bar{\vec{V}}_M$：同周期全市场报告平均向量。  | Cohen et al. (2020)          |
| **II. 行为偏差**   | **4. 自利归因** (Self-Serving)    | $SS_i = \sum_{w \in D_{1st}} TF\text{-}IDF_{w,i}$                                             | $D_{1st}$：第一人称代词词典（如“我”、“我们”）。                      | Hribar & Yang (2016)         |
|                | **5. 过度自信** (Overconfidence)  | $OC_i = \frac{N_{str,i} - N_{wk,i}}{N_{str,i} + N_{wk,i}}$                                    | $N_{str}$：强语气词数；$N_{wk}$：弱语气词数。                     | L&M (2011)                   |
|                | **6. 时间视域** (Time Horizon)    | $Horizon_i = \frac{Count(D_{short})_i}{Count(D_{long})_i}$                                    | $D_{short}$：短期视域词典；$D_{long}$：长期视域词典。               | 胡楠等 (2021)                   |
| **III. 代理问题**  | **7. 策略性模糊** (Obfuscation)    | $Fog_i = 0.4 \times (ASL_i + PHW_i)$                                                          | $ASL$：平均句长；$PHW$：复杂词（三音节及以上）占比。                     | Bloomfield (2008)            |
|                | **8. 责任承担** (Accountability)  | $Acc_i = \frac{\sum TF\text{-}IDF_{int,i}}{\sum TF\text{-}IDF_{ext,i}}$                       | $int$：内部归因词（能力）；$ext$：外部归因词（运气、环境）。                 | Sydserff & Weetman (2002)    |
|                | **9. 话题转移** (Focus Shifting)  | $Shift_i = \frac{\vec{V}_{i,t} \cdot \vec{V}_{i,t-1}}{\|\vec{V}_{i,t}\| \|\vec{V}_{i,t-1}\|}$ | $\vec{V}_{i,t}$：本期报告向量；$\vec{V}_{i,t-1}$：该基金上期报告向量。 | Lang & Stice-Lawrence (2015) |
